どうぶつのもり

Javascript(Jquery)を用いてアニメーションを制作するという課題にて制作した的あてゲームです。

URL

./works/js-works01/index.html

使用技術

HTML / CSS / JavaScript / jQuery

使用アプリ

VisualStudioCode / Illustrator / Photoshop

作品紹介

「はじめる」ボタンを押すと画面内のモンスターが動き出すので、それをクリックして退治するというコンセプトの簡単なゲームです。

工夫したところ

6秒ごとに50%の確率で表示/非表示が切り替わる仕組みを考え、レアモンスターの演出という形で実装しました。
また、よりゲーム性が高まるように制限時間を設け、タイマーがゼロになるとゲーム画面内でクリックができなくなるようにしました。

制作時間

デザイン:1時間
画像編集:1時間
コーディング(プログラミング):9時間
