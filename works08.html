使用技術

HTML / CSS / JavaScript / Node.js / pug / Bootstrap / Prisma / Docker

使用アプリ

VisualStudioCode / Git / Github / Render

作品紹介

web ページのタイトルとURL を保存することを目的として、認証・投稿・保存・削除の機能を持ったアプリケーション

なぜ作ろうと思ったか

就職活動において利用する求人サイトが増えて求人を管理するのが難しくなったこと、また職業訓練校卒業後も学習を続けるなかで、在校中に作成したGoogle Chrome拡張機能を下地に発展させようと思い作成。

工夫したところ

入力フォームのバリデーションや、投稿の並び替え、認証前トップページの実装など

感想

Node.js でHTTP サーバー構築、ルーティング処理、フロント側へのデータ受け渡しなどのバックエンド側の一連の処理を学べたことが良かったです。
一方でユーザーの追加機能や、ページネーションなどの一般的な機能を実装できていなかったり、データベースの設計など多くの課題を感じたので、これからも学習を継続したいと思います。
